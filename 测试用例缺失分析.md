# 测试用例缺失分析

## 1. TBR vs Non-TBR 模式切换

| 缺失测试 | 说明 |
|---------|------|
| `test_TBR_ModeSwitch` | 验证 `setTBREnabled(true/false)` 两种模式输出一致 |
| `test_TBR_vs_NonTBR_SameOutput` | 相同场景在 TBR 和 non-TBR 模式下应产生相同结果 |

## 2. TilingStage 单元测试

| 缺失测试 | 说明 |
|---------|------|
| `test_TilingStage_TriangleBinning` | 验证三角形正确分配到对应 tiles |
| `test_TilingStage_MultiTileTriangle` | 跨越多个 tiles 的大三角形 |
| `test_TilingStage_TriangleAtTileBoundary` | 顶点在 tile 边界上 |
| `test_TilingStage_DegenerateTriangle` | 退化三角形（面积为0） |
| `test_TilingStage_TriangleAtScreenEdge` | 三角形在屏幕边缘 |

## 3. Per-Tile Rasterization

| 缺失测试 | 说明 |

| `test_Rasterizer_TriangleSpansMultipleTiles` | 跨 tile 三角形的正确渲染 |
